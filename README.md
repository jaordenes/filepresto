# FilePresto

**Free online file converter - Images, PDF, Video, Audio, Documents, Ebooks**

Live site: https://jaordenes.github.io/filepresto/

---

## English

FilePresto is a free, browser-based file conversion tool. No installation required - convert images, PDFs, videos, audio files, documents and ebooks directly in your browser, for free.

### Features

- Image conversion: JPG, PNG, WebP, GIF, SVG, HEIC, BMP and more
- PDF tools: Compress, rotate, merge, convert PDF to/from images
- Video conversion: MP4, AVI, MKV, WebM, MOV and more
- Audio conversion: MP3, WAV, FLAC, OGG, AAC and more
- Document conversion: DOC, DOCX, ODF, TXT and more
- Ebook conversion: EPUB, MOBI, AZW3, PDF and more
- Multilingual: English, Spanish and Portuguese
- No registration required - completely free
