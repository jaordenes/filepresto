# Contributing to FilePresto / Contribuyendo / Contribuindo

## English

Thank you for contributing! Please follow these guidelines:

**Reporting Bugs:** Open a GitHub issue with clear steps to reproduce.

**Feature Requests:** Open a GitHub issue with [Feature Request] prefix.

**Code Contributions:**
1. Fork the repo and create a feature branch
2. Make changes maintaining EN/ES/PT trilingual structure
3. Use semantic HTML5, maintain accessibility (ARIA, landmarks)
4. Open a Pull Request with a clear description

**File Structure:** Maintain trilingual structure for all new pages.
